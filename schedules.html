Seat Class Guide

Understanding what each class offers

First Class

Air-conditioned, reserved reclining seats. Complimentary meals on long routes. Most comfortable option.

Second Class

Reserved padded seats, ceiling fans, good legroom. Ideal for medium to long journeys.

Third Class

Unreserved seats. Budget-friendly option for shorter journeys. First come, first seated.
